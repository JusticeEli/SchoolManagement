# SchoolManagement
This is the [link](https://play.google.com/store/apps/details?id=com.justice.schoolmanagement) to the app in google play store

Mobile app that makes it easier to maintain records of a school, has a section for managing.
* teachers
* parents
* students
* classes
* subjects 
* results


Teachers can 
- easily update marks of students remotely 
-  contact their respective parents
-  Schedule Real-Time video calls and chats .

## Tech Stack
- Kotlin
- Firebase Authentification
- Cloud Firestore
- Firebase Storage
- Firebase ML
- Glide
- Dagger Hilt
- Material Designs
- Jetpack Components
- Coroutines
